🔐 Daten-Migration: Verschlüsselung

⚠️ Wichtig: Dieses Tool verschlüsselt alle bestehenden Antworten in der Datenbank. Dies sollte nur EINMAL ausgeführt werden, nachdem die Verschlüsselungsfunktion implementiert wurde.
Eingeloggt als:

Was wird gemacht?

  • Alle Antworten in der Collection antworten werden geladen
  • Für jede Antwort mit unverschlüsselten Daten (kindname, email):
    • Kindname und E-Mail werden verschlüsselt
    • Ein Hash vom Kindname wird erstellt (für Gruppierung)
    • Die verschlüsselten Felder werden zur Antwort hinzugefügt
  • Die originalen Felder kindname und email bleiben erhalten (als Fallback)
Hinweis: Nach erfolgreicher Migration können Sie in einem weiteren Schritt die unverschlüsselten Felder optional entfernen. Diese Migration fügt nur die verschlüsselten Felder hinzu.
